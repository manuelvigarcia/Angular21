Desarrollo de aplicaciones de cliente ligero que sólo requieren el navegador.

Separa la visualización HTML de la funcionalidad.

Usa TypeScript (orientado a objetos)

Se desarrolla completamente contra un servidor, no es ejecutable directamente en el navegador y luego se compila a JavaScript.

Se basa en el patrón MVC
	- Componente 
			- vista para presentación al usuario (plantilla HTML)
			- Controlador, llamado "clase componente"
	- Modelo, llamado "servicio.


Vamos a usar Angular 21

Se crea aplicación desde el terminal de VS Studio Code con
	> ng new <nombreAplicacion>

Para crear un workspace donde crear aplicaciones, para que no se duplique con cada una todos los paquetes:
	> ng new nombre-workspace --create-application=false
	
Y luego ya se crea cada aplicación con
	> ng generate application nombre-app

Usando ` como delimitador de cadenas puede usarse ${<nobre_variable} dentro de una cadena para insertar valores computados
	`Te llamas ${this.nombre}  y tienes ${this.edad} +  años` en lugar de
	"Te llamas " + this.nombre + " y tienes " + this.edad + " años";
	
	
La sintaxis requiere poner ';' al final de cada sentencia. Pero el compilador no lo exige. 	
	
en el index.html principal es donde se añaden los links de bootstrap, por ejemplo, dentro del <head\>
	<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css"> 
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>

Cuando se envían objetos en el HTTP request (POST) se puede enviar el objeto directamente, indicando que se codifica como json:
		heads=heads.set("Content-Type","application/json");
		return this.http.post<void>(url,cliente,{"headers":heads});

El resultado del servicio HTTP, se pueden recibir los encabezados si en la invocación se indica 'observe:"response"'
		buscar(tematica:string):Observable<any>{
			return this.http.get(this.urlBase+"buscar",
				{params:{tematica:tematica},observe:"response"});
		}
